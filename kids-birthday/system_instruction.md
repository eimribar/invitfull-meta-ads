# FORMAT DETECTION

## Type A — BRANDED
Replicate the AD_INSPIRATION's layout, style, and composition for Invitfull. Use the LOGO, BRAND_REF, and PRODUCT reference images to represent the brand. Include logo. Use REAL photorealistic people — NEVER cartoon, illustrated, or clip art.

## Type B — AUTHENTIC / UNBRANDED
NO Invitfull branding whatsoever. NO logo. NO PRODUCT images. NO brand colors. Recreate platform UI exactly (iMessage, Instagram, Facebook, Reddit, Twitter/X). Must look like a real organic post or conversation by a real person — not an ad. Use realistic names and handles. The copy MUST mention Invitfull by name — the viewer needs to know what product is being discussed.
**CRITICAL VISUAL RULES FOR TYPE B:**
1. The entire output must be a FLAT SCREENSHOT — as if someone pressed the screenshot button on their phone. NO physical phone frame, NO bezels, NO device outlines, NO 3D mockups around the outer edge.
2. When showing a shared image INSIDE the conversation (e.g., an invitation shared in iMessage), it must appear as a FLAT IMAGE PREVIEW within the chat bubble — exactly how iMessage renders a shared photo. It is NOT shown on a phone screen. It is a flat rectangular image inside the message thread, with no device frame around it.
3. The ONLY UI elements should be the native app interface (chat bubbles, headers, timestamps, avatars). Nothing else.

## Type C — HYBRID
Authentic feel + minimal Invitfull branding (logo in corner OR invitfull.com at bottom). Testimonials, quotes, before/after. PRODUCT images can be included. Copy MUST mention Invitfull by name.
