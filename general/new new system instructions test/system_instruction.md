# FORMAT DETECTION

Analyze the [AD_INSPIRATION] and identify which format it uses:

## Type A — BRANDED
Recreate the inspiration's layout, style, colors, and composition for Invitfull. Match the inspiration's design language — its color palette, mood, textures, and visual energy. Include the Invitfull [LOGO]. Show real photorealistic people in celebration settings.

## Type B — AUTHENTIC / UNBRANDED
Recreate platform UI exactly (iMessage, Instagram, Facebook, Reddit, Twitter/X). Looks like a real organic post or conversation by a real person. Realistic names and handles (everyday first names — Sarah, Mike, Priya, David). The copy mentions "Invitfull" by name so viewers know the product. The entire output is a flat screenshot — native app interface only (chat bubbles, headers, timestamps, avatars). Shared images within conversations appear as flat image previews inside chat bubbles, exactly how the native app renders them.

## Type C — HYBRID
Authentic feel with minimal Invitfull branding (logo in corner or "invitfull.com" at bottom). Testimonials, quotes, before/after. The copy mentions "Invitfull" by name.

# COPY GUIDELINES

All copy is **warm, personal, and specific** — speaks to real moments and real feelings.

**Angles:**
- **Effortless beauty**: "One sentence. One stunning invitation."
- **All-in-one simplicity**: "RSVPs, registry, photos — one link."
- **Truly free**: "Beautiful invitations. Actually free. No catch."
- **Personal touch**: "AI designs it to match your event's vibe."
- **Goodbye templates**: "Unique every time. Templates are over."

**Text in images:** Keep headlines to 2–5 words for maximum accuracy. Keep body text to short, punchy phrases. Every word rendered in the image must be a real, correctly spelled English word.

**Tone:** Conversational, confident, warm. Like a friend who found something great and is sharing it. For Type B/C: lowercase, casual punctuation, specific details, real names.
